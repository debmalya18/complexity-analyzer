# ⚡ Code Complexity Analyzer

A web app that analyzes any code and instantly returns Big O time and space complexity — works with any programming language.

🔗 **Live Demo:** [complexity-analyzer-tau.vercel.app](https://complexity-analyzer-tau.vercel.app)

## 🚀 Features
- Paste any code — Java, C++, Python, JavaScript, Go, Rust and more
- Instantly get Time Complexity and Space Complexity
- Detects O(1), O(n), O(n²), O(n³), O(log n), O(2^n), O(n log n)
- Clean dark themed UI
- Works 100% offline — no API needed

## 🛠️ Tech Stack
- React
- CSS3 (custom dark theme)
- Deployed on Vercel

## 📸 How to Use
1. Open the live link
2. Paste your code in the editor
3. Click **Analyze Complexity**
4. Instantly see your Big O result!
